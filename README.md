Sistema de Automação Python - v6.1 (Módulo Caçador)
Este projeto nasceu da necessidade de automatizar tarefas repetitivas no Windows e organizar a busca de arquivos de forma inteligente.

 O que ele faz?
Módulo Caçador: Realiza buscas profundas no sistema de arquivos para localizar itens específicos de forma rápida.

Interação com Interface: Utiliza automação visual para interagir com janelas e elementos do sistema.

Execução Facilitada: Inclui um arquivo .bat que permite rodar o sistema com apenas dois cliques, sem precisar abrir o editor de código.

 Tecnologias Utilizadas
Python 3: Linguagem base para toda a lógica e processamento.

PyAutoGUI: Para controle de periféricos e automação de interface.

Git & GitHub: Para controle de versão e documentação profissional do código.
Como funciona?
O sistema foi projetado para ser modular. O coração do projeto é o sistema.py, que gerencia as chamadas de funções, enquanto scripts auxiliares cuidam da parte pesada de busca e captura.
